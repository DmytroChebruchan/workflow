## Getting Started

### Installation

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the App

1. Start the app:
   ```bash
   python -m main
   ```

### Documentation

- Access the FastAPI documentation page at: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
